Things I also do that I must stop doing
Because balance matters

I spend too much time on Netflix, YouTube, and Reddit. I will make a plan that will allow me greater control of myself starting now.
Basically, my plan is that at most I can only use two of these things in any given week. So if I use Netflix and YouTube, I cannot use Reddit in that same week. Furthermore, in order to use any of these things, I must do a certain amount of productive things first. So if I work on this website for two hours, I can waste half an hour doing any of the above things.
